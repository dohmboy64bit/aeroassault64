ROM dumps in this folder are intentionally not tracked by git.

Canonical path for tooling is the repo root: `roms/afa.n64.us.z64` (see `roms/README.txt`).
If you keep a working copy here, add patterns under `Docs/Rom/` in the root `.gitignore` already ignore `*.z64` / `*.n64` / `*.v64`.

SHA1 recorded in `config/splat.yaml` was computed from a local `afa.n64.us.z64`; confirm it matches your intended USA dump before relying on it for splat.
